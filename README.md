# HR ANALYTICS
This project focuses on analyzing Human Resources (HR) data to derive meaningful workforce insights and present them through an interactive web-based application. It involves data cleaning, exploratory data analysis (EDA), calculation of key HR metrics, and visualization to understand employee trends, performance, and attrition patterns.

The project is built using Python and Jupyter Notebook, with an integrated web interface to make insights easily accessible for HR decision-making.

Tech Stack: Python, Pandas, NumPy, Matplotlib/Seaborn, Jupyter Notebook
